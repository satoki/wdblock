# WdBlock
Blocks annoying donation ads on Wikipedia.
For Firefox,Chrome.
ウィキペディアの寄付広告をブロックする。
FirefoxとChrome用。

## Demo

Please use after donating.
寄付した後で使ってください。

![After](https://github.com/satoki/wdblock/blob/images/02.png)
